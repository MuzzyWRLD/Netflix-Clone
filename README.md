# 🎬 Netflix-Clone Projekt

Dieses Projekt ist mein 15-tägiger Lernpfad in die Full-Stack-Webentwicklung mit Django. Es umfasst ein Frontend-Design, eine dynamische Datenbank-Anbindung und ein sicheres Benutzer-Authentifizierungssystem.

## 🛠 Technologien
- **Backend:** Python, Django 6.0.6
- **Frontend:** HTML5, CSS3, JavaScript
- **Datenbank:** SQLite3
- **Versionierung:** Git/GitHub

## 🚀 Installation & Start
1. Repository klonen: `git clone https://github.com/MuzzyWRLD/netflix-clone.git`
2. Virtuelle Umgebung erstellen: `python3 -m venv blase`
3. Umgebung aktivieren: `source blase/bin/activate`
4. Server starten: `python3 manage.py runserver`

## 🚨 Troubleshooting
- **Server findet Datei nicht:** Server neu starten (`Strg+C` & `runserver`).
- **TemplateDoesNotExist:** Pfad prüfen, ob Datei im richtigen `templates/`-Unterordner liegt.
- **Git Push abgelehnt:** `git pull --rebase origin master` vorher ausführen.
